üì¶ Step 1: Install the Extension

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked"
  4. Select the d365-automation-extension folder
  5. Extension icon should appear in Chrome toolbar!

Note: You'll see a warning about icons - that's okay for now. See icons/README.md for creating icons.

üéØ Step 2: Your First Workflow (5 Minutes)

Scenario: Add a charge code translation

  1. Navigate to D365FO
    • Open your charge codes form
    • Filter to a charge code (e.g., "AUF-DE")
  2. Click extension icon
    • Should show "Connected to D365FO" (green indicator)
    • If not, refresh the page
  3. Go to "Inspector" tab
    • Click "Start Inspector"
    • Extension window will close
    • Hover over the "Translations" button
    • It will highlight in blue!
    • Click it
  4. Reopen extension
    • You'll see the element details were captured!
    • Control Name: LanguageTxt
    • Display Text: "Translations"

üì∏ Visual Guide: Element Picker

üé¨ How Element Picker Works

When you click "üéØ Pick from Page":

  1. Extension window closes
  2. Blue overlay appears on D365 page
  3. As you hover, elements highlight with blue border
  4. Click any element to select it
  5. Extension automatically captures:
    • Control name (technical ID)
    • Display text (human-readable)
    • Field type (if applicable)

üí° Pro Tips

  • Test with 1 row first! Validate your workflow before processing 100+ records.
  • Use descriptive names: Instead of "Workflow 1", use "Add Charge Translations"
  • Add wait steps: After clicks that open dialogs, add 800ms wait
  • Check the console: Open DevTools (F12) to see execution logs
  • Export your workflows: Save as JSON for backup/sharing
  • Timing issues? Go to Settings tab and increase delays

üîß Troubleshooting

Extension shows "Not connected"

  • Make sure you're on a D365FO page
  • Refresh the page
  • Reopen the extension popup

Element picker not highlighting anything

  • Close and reopen extension
  • Try clicking "Start Inspector" again
  • Check browser console for errors

Values not being set correctly

  • Increase delays in Settings tab
  • Check that field mappings match your data columns exactly
  • Verify field type (enum vs text) in Inspector tab
